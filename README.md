# Supply Chain Inventory Analysis using Power BI

## Project Description
This project focuses on analyzing inventory and supply chain data to identify
stock shortages, excess inventory, and optimize inventory management decisions
using Power BI.

## Objectives
- Identify products below reorder point
- Analyze stock levels using descriptive analysis
- Classify inventory as Low, Optimal, or Excess
- Identify slow-moving products
- Monitor inventory turnover rate
- Design an interactive dashboard to track stock health

## Tools & Technologies Used
- Power BI
- DAX
- Excel / CSV Dataset
- GitHub

## Key Analysis Performed
- Univariate analysis on stock levels
- Warehouse-wise stock shortage analysis
- Inventory turnover calculation
- Stock status classification using DAX
- Identification of products requiring immediate replenishment

## Dashboard Features
- KPI cards for stock metrics
- Warehouse-wise stock analysis
- Product-level inventory status
- Visual indicators for low stock alerts

## Outcome
The dashboard helps reduce holding costs, prevent stock-outs,
and improve inventory decision-making.
